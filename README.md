# MoistureTransport

Authors:

    - Luo, Likun
    - Holzner, Peter

A moisture transport simulation tool.

## Usage

### Source version

1. Clone the repository and use

2. Run using

    ```bash
    ./python main.py
    ```

### Release/Binary version

1. Download the binary

2. Run using

    - Windows:

        ```bash
        ./moisture_transport.exe
        ```

    - Linux:

        ```bash
        ./moisture_transport.exe
        ```
